# Camera Robot – University Team Project

This repository is a showcase copy of a semester team project at TU Berlin from Summer 2019.

The original repository was developed collaboratively and is private. This public version is intended to document the project structure, technologies used, and implementation approach for portfolio/recruiting purposes.

The project combined:
- Arduino firmware for servo/sensor control
- A Processing interface for camera/robot control
- A small Python/Tkinter GUI prototype
